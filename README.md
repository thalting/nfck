# neof*ck
A programming language based on the brainf*ck specification.

## Usage
    $ nfck source.nfck

## brainf*ck equivalents

| neof*ck | brainf*ck |
| --- | ----------- |
| ) | > |
| ( | < |
| * | + |
| / | - |
| ! | . |
| ? | , |
| { | [ |
| } | ] |

## Build
    $ make

## Translate brainf*ck into nfck code
    $ translator code.bf code.nfck
